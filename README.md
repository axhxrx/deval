# deval

Developer environment evaluation and benchmarking tool.

## Purpose

- Benchmark containerized (Incus/LXC/Docker), VM, and native dev environments
- Compare performance across different machines and setups
- Track performance metrics over time

## Benchmarks

- File I/O Ops (many small files)
- Compilation performance (Deno, Bun, tsc, Rust)
- Package manager Ops (pnpm install, npm ci)
- Real-world build scenarios
